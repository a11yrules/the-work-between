## Introduction

I'm committed to accessibility. This website should work for everyone. Simple as that.

This statement covers my current and ongoing accessibility efforts, how I test this site,
contact information, and current conformance status. There are no known issues at this time.

## How I Test

I use a mix of methods to evaluate accessibility.

**Automated testing** catches common issues like missing alt text, low contrast, and 
structural errors. It's a useful first pass, but it only finds a fraction of potential problems.

**Manual testing** goes further. I check keyboard navigation, screen reader behavior, 
focus management, and other issues automated tools miss.

**Functional testing** verifies that interactive elements work as expected across 
assistive technologies and browsers.

**Usability testing with disabled users** disabled people relying on assistive technology test the site directly and give feedback on their actual experience.
